# САУ лифтом конструкции патерностер
Данный проект является учебным. Он создан для выполнения лабораторной работы номер 6 по курсу ТАУ и представляет собой папку проекта MatLab.

## Цели и задачи
Данный проект преследует цель создания системы автоматического управления лифтом конструкции патерностер. Это включет в себя:
* Создание и структуризацию директории проекта
* Настройку проекта
* Создание модели системы
* Выбор и настройку архитектуры корректирующего устройства
* Создание тестов для проверки удовлетворения уловиям

Управление лифтом осуществляется засчет подачи на вход двух сигналов: периода обращения и амплитуды движения. Управляющий сигнал формируется путем деления времени периода на 4 части, две из которых представляют собой остановку на высоте 0 и высоте, равной амплитуде, а остальные две - подъем и спуск с постоянной скоростью.

## Тесты
#### AccuracyTest
Данный тест сравнивает вид выходного сигнала (положение) и сформированного входного (обрезанная пила). Прохождение теста счиатется успешным, если в любой момент времени отклонение составляет не более 5% от входной амплитуды. С данными настройками системы управления тест проходит проверку при моменте нагрузки равном 0.25, амплитуде 10 и периоде 10.
